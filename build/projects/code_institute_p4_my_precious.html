Overview

My Precious helps people reunite with lost items. Authenticated users can create posts (lost or found), add photos and locations, browse & search posts, and chat via private messages.

Features

  • User accounts with authentication & profiles
  • CRUD for posts with images and location
  • Search & filters for lost/found listings
  • Private messaging between users
  • Deployed to Heroku; assets via Cloudinary; DB hosted on ElephantSQL
